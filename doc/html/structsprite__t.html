The position of the left-bottom coordinate of the sprite.

unsigned int sprite_t::position_uniform

The OpenGL uniform of the position, used by the vertex shader.

We use a uniform because the position of the sprite could vary. The uniform allows us to change a vertex buffer variable from the C code (CPU memory).
