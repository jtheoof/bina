Detailed Description

Main GLUT file used for oses that support GLUT.

It should be possible to write a very simple wrapper around GLUT for devices that do not support it.

See Also
http://freeglut.sourceforge.net/docs/android.php
http://en.wikibooks.org/wiki/OpenGL_Programming/Android_GLUT_Wrapper

(GLUT does not seem to have been ported to iOS).
