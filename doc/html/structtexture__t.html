unsigned int texture_t::unit

Specifies which texture unit to make active.

The number of texture units is implementation dependent, but must be at least 8. texture must be one of GL_TEXTUREi, where i ranges from 0 to (GL_MAX_COMBINED_TEXTURE_IMAGE_UNITS - 1). The initial value is GL_TEXTURE0.
